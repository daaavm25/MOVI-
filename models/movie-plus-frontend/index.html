Servidor API & GitHub Pages

Configura la URL del servidor backend. Necesario si despliegas la interfaz en GitHub Pages u otro host estático separado del servidor.

💡 Cómo desplegar en GitHub Pages
  1. 1. Despliega el backend en Railway, Render, Heroku u otro servicio externo.
  2. 2. Copia la URL pública del backend y pégala en el campo de arriba. Guarda y recarga.
  3. 3. Sube el contenido de la carpeta movie-plus-frontend/ a un repositorio de GitHub.
  4. 4. Ve a Settings → Pages, selecciona la rama y carpeta raíz / (root).
  5. 5. GitHub Pages publicará la interfaz automáticamente en minutos.
